    Einführung in maschinelles Lernen

    Seit 2012 ist eine Form der digitalen Mustererkennung besonders in den Fokus der interessierten Öffentlichkeit geraten. Neuronale Netzwerke bestehen aus Knoten und Verbindungen zwischen diesen Knoten. Die Knoten werden dabei als Neuronen bezeichnet und ihre Verbindungen als Gewichtungen. Um dem Netzwerk Struktur zu geben, werden die Neuronen in Ebenen geschichtet. Typischerweise wird diese Art von Netzwerken aus einer Eingabeschicht, folgend eine oder mehrere verdeckte Schichten, und einer Ausgabe Schicht geformt. Die verdeckten Schichten werden als verdeckt bezeichnet, weil sie nur über die Verbindung zur Eingabe- und Ausgabeschicht sichtbar sind. Aus der Vielzahl von verdeckten Schichten leitet sich der englische Begriff des Deep Learnings ab. Eben weil das Netzwerk durch die Anzahl der verdeckten Schichten an Komplexität zunimmt.

    Ein kurzes Beispiel soll anhand dieser Programmanweisungen erläutert werden. MNIST Zahlenerkennung

    Im Beispiel wird das beliebte Machine Learning Framework 'Google Tensorflow' benutzt. Nachdem das Modul importiert worden ist, wird ein Trainings- und Testdatensatz, die Zahlensammlung MNIST, geladen. Dieser Datensatz besteht aus 60.000 Trainingsbildern, die handgeschriebene Zahlen darstellen, und 10.000 Testbildern, die zur Verifikation der Klassifizierungsentscheidung durch das Neuronennetzwerk verwendet werden. Die Größe der Bilder beträgt (nur) 28x28 Pixel. Wenn ein Trainingsbild aus mehr Pixeln besteht, steigt die Anzahl der benötigten Rechenoperationen und der HPC Client (Workstation, Cloud) muss unter sonst gleichen Bedingungen deutlich länger rechnen. Lange und hohe Auslastung von CPUs und GPUs bedeutet auch freilich, dass ein hoher Stromverbrauch zu erwarten ist. Das muss sollte daher bei der Wahl der Trainingsdaten bedacht werden.

    In einem nächsten Schritt wird die Abstraktionsschicht 'Keras' (=Keras API von François Chollet) verwendet, welche die Benutzung von Tensorflow vereinfacht. Trainungs- und Testdaten werden normalisiert, was bedeutet, dass aus den Vektoren von Zahlen zwischen 0-255, die je eine Zahl (=ein Trainingsbild) darstellen, Fließkommawerte zwischen 0 und 1 erstellt werden. Es folgt die nähere Definition des Modells, das als 'Sequential' festgelegt wird. Zwei verdeckte Ebene enthalten 128 Knoten, denen eine Aktivierungsfunktion zugeordnet wird (relu, Rectified Linear Unit). Die Aktivierungsfunktion kann als ein Wahrscheinlichkeitsmaß verstanden werden, welches beschreibt, ob ein Neuron im Netzwerk aktiv ist und falls ja, wie stark. Ein Neuron soll genau dann besonders aktiv sein, wenn es eine Eigenschaft, die zur Klassifizierung der Eingabedaten von Bedeutung ist, detektiert hat.

    Als Ausgabeschicht wird eine Schicht bestehend aus 10 Neuronen mit der Aktivierungsfunktion Softmax gewählt. Softmax liefert eine Wahrscheinlichkeit für die Erkennung einer bestimmten Klasse aus einer Menge an Klassen, entscheidet also letztlich, welche Klassifizierung des neuronale Netzwerk geben wird. Jedes Neuron steht für eine der 10 Zahlen, die der MNIST Datensatz enthält.

    Nun kann die Verarbeitung der MNIST Bilder gestartet werden. Dazu wird das Modell mit einem zu wählenden Optimierer (optimizer), einem Verlustmaß ('loss') und einer Metrik ('metric') kompiliert. Der Signatur der Methode fit werden die Trainungsdaten und eine Epochenzahl (Anzahl Trainingsdurchläufe) übergeben. Als Ergebnis wird ein Genauigkeitmaß geliefert, das beschreibt in wieviel Fällen das Modell eine richtige Klassifizierung vorgenommen hat, zB im Fall des vorliegenden Modells in 97,58% der Fälle. In 2,42% der Testklassifizierungen liegt das Modell jedoch falsch. Die Idealvorstellung ist natürlich ein Klassifizierungsfehler von Null.

    Welche Hardware brauche ich für Machine Learning?

    Als Beginner braucht man gar keine spezielle Hardware. Die verwendeten Frameworks arbeiten sowohl mit Prozessoren (CPU), als auch mit Grafikkarten (GPU). GPUs sind jedoch deutlich schneller im Berechnen eines neuronalen Netzwerks. Das gezeigte Beispiel (s.o.) belegt ca. 500 MB Arbeitsspeicher. Daher sollte man auch als Einsteiger mindestens 4-8 GB Arbeitsspeicher zur Verfügung haben.

    Sollen nach dem Erlernen der Grundlagen größere Netzwerke erstellt oder größere Datensätze verarbeitet werden, kann eine Nvidia GTX 1060 (6 GB) eine gute Wahl sein. Der Preis beträgt ca. 200-300 Euro, je nach Hersteller. Alles preislich darüber kann als der ambitionierte Bereich betrachtet werden. Wer sich hier betätigen möchte, kann eine Nvidia GTX 1080 (~ 520 €), GTX 1080ti (~760 €) erwerben, die hohe Rechenleistung und viel Speicher (8-11 GB) bieten. Ab September kann auch die neuste Generation an Nvidia GPU verwendet werden (Nvidia RTX Turing Architektur, RTX 2080, RTX 2080ti (~ 1.300 €).
