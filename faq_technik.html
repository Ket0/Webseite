F: Wie wird ein öffentlicher SSH Schlüssel aus ssh-add zu Git kopiert?

A: Der SSH Schlüssel muss über

$ ssh-keygen -t rsa -b 4096 -C "your_email@example.com" erzeugt werden. Ein Passwort String wird gewählt und gespeichert.

mit $ ssh-add ~/.ssh/id_rsa wird der generierte SSH Schlüssel im SSH Konfigurationsprogramm ('ssh-agent') gespeichert. Anschließend muss der Schlüssel aus der Datei des SSH Konfigurationsprogramms in das richtige Clipboard ('Zwischenspeicher') kopiert werden. Weil es mehrere Clipboards gibt, muss ausdrücklich das korrekte Clipboard benutzt werden bzw. der SSH Schlüssel muss in dieses kopiert werden. Dazu wird folgede Vorgehensweise verwendet: Das Programm 'xclip' muss in der Shell installiert werden, wenn es nicht schon vorhanden ist.

sudo apt install xclip dann cat ~/.ssh/id_rsa.pub | xclip -selection -c
